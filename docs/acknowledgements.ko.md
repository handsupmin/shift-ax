# Shift AX 감사와 샤라웃

**English version:** [acknowledgements.md](./acknowledgements.md)

Shift AX는 여러 훌륭한 레포의 아이디어, 워크플로우, 구현 패턴에서 영향을 받았습니다.

이 문서는 Shift AX에 영향을 준 프로젝트들에 대한 직접적인 샤라웃입니다.

세부 채용 범위와 경계는 아래 문서를 참고하세요.
- `docs/roadmap/source-adoption-backlog.md`

## 샤라웃

- [oh-my-codex](https://github.com/Yeachan-Heo/oh-my-codex.git)
  - multi-agent workflow framing, runtime coordination, durable state 개념
- [oh-my-claudecode](https://github.com/Yeachan-Heo/oh-my-claudecode)
  - Claude 중심 workflow integration, session guidance, operator ergonomics
- [agent-skills](https://github.com/addyosmani/agent-skills)
  - skill anatomy, context hierarchy, spec-first, review discipline
- [agent-orchestrator](https://github.com/ComposioHQ/agent-orchestrator)
  - lifecycle, reaction, observability 개념
- [mempalace](https://github.com/milla-jovovich/mempalace)
  - onboarding, room/entity detection, long-term memory support 개념
- [claw-code](https://github.com/ultraworkers/claw-code)
  - doctor-first 운영 관점과 실용적인 CLI ergonomics

## 메모

Shift AX는 이들 중 하나를 그대로 복제하려는 프로젝트가 아닙니다.

목표는 AX에 익숙하지 않은 팀에게 실제로 도움이 되는 강한 아이디어만 뽑아서, 더 단순한 docs-first request-to-commit 시스템으로 다시 만드는 것입니다.
