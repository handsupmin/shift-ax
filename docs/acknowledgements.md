# Shift AX Acknowledgements

**한국어 버전:** [acknowledgements.ko.md](./acknowledgements.ko.md)

Shift AX was shaped by ideas, workflows, and implementation patterns from several excellent repos.

This page is a direct shout-out to the projects that influenced the product.

For the detailed adoption choices and boundaries, see:
- `docs/roadmap/source-adoption-backlog.md`

## Shout-outs

- [oh-my-codex](https://github.com/Yeachan-Heo/oh-my-codex.git)
  - strong multi-agent workflow framing, runtime coordination, and durable state ideas
- [oh-my-claudecode](https://github.com/Yeachan-Heo/oh-my-claudecode)
  - Claude-focused workflow integration, session guidance, and operator ergonomics
- [agent-skills](https://github.com/addyosmani/agent-skills)
  - skill anatomy, context hierarchy, spec-first and review discipline
- [agent-orchestrator](https://github.com/ComposioHQ/agent-orchestrator)
  - lifecycle, reaction, and observability concepts for agent execution systems
- [mempalace](https://github.com/milla-jovovich/mempalace)
  - onboarding, room/entity detection, and long-term memory support concepts
- [claw-code](https://github.com/ultraworkers/claw-code)
  - doctor-first operational thinking and practical CLI workflow ergonomics

## Note

Shift AX does not try to be a copy of any one of these repos.

The goal is to take the strongest ideas that help teams with low AX familiarity and rebuild them into a simpler docs-first request-to-commit system.
