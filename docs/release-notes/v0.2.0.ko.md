# Shift AX v0.2.0

**English version:** [v0.2.0.md](./v0.2.0.md)

## 요약

Shift AX v0.2.0은 workflow 중심 source checkout 도구에서 한 단계 더 나아가, 실제로 설치해서 쓰는 **product shell** 방향을 강화한 버전입니다.

이번 릴리즈의 핵심은:

- 더 쉬운 설치
- 더 쉬운 첫 실행 온보딩
- 플랫폼 자연스러운 진입점
- 그리고 그 아래에 유지되는 동일한 docs-first request-to-commit 코어

## v0.2.0에 포함된 것

- 대화형 셸 진입점:
  - `ax --codex`
  - `ax --claude-code`
  - `shift-ax --codex`
  - `shift-ax --claude-code`
- shared context artifact가 없을 때 자동 first-run onboarding
- 첫 실행 언어 선택:
  - English
  - Korean
- guided onboarding 질문지:
  - business context
  - primary workflow
  - no-guessing policy areas
  - risky domains
  - architecture summary
  - important paths
  - internal terms / aliases
  - default verification commands
- locale / preferred platform 저장용 `.ax/project-settings.json`
- Codex / Claude Code 세션 안에서 slash / dollar command를 product-shell command로 해석하도록 하는 bootstrap guidance
- npm 공개 설치 패키징
- one-command global installer script

## 검증 포인트

- 전체 자동 test / build 검증
- 셸 런처 테스트:
  - Codex shell
  - Claude Code shell
  - first-run language + onboarding flow
- packed tarball 기준 global install smoke test
- npm publish 완료:
  - `@handsupmin/shift-ax@0.2.0`

## 그대로 유지된 것

request-to-commit control plane의 핵심 원칙은 그대로 유지됩니다.

- docs-first context resolution
- planning interview
- human plan review
- implementation 전 policy sync
- structured review lanes
- Lore commit finalization

## 운영자 관점 결론

v0.2.0은 Shift AX가 더 이상 내부 CLI wrapper처럼 느껴지지 않고, 실제 product shell처럼 느껴지기 시작하는 버전입니다.

팀이 아래를 원한다면:

- 플랫폼 자연스러운 진입점
- 자동 first-run setup
- 덜 거친 onboarding 경험

v0.2.0을 기본선으로 보는 것이 맞습니다.
