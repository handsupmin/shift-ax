# Shift AX v0.1.0

**English version:** [v0.1.0.md](./v0.1.0.md)

## 요약

Shift AX v0.1.0은 첫 번째 완성형 request-to-commit control plane입니다.

대상은 AX에 익숙하지 않은 팀이며, 아래가 필요한 팀을 위한 버전입니다.
- document-first grounding
- 눈에 보이는 planning / review gate
- 추적 가능한 topic artifacts
- 복잡성보다 안전한 기본값

## v0.1.0에 포함된 것

- domain / policy / architecture context onboarding
- base-context discovery 및 glossary seeding
- request를 topic/worktree artifact로 bootstrap
- planning interview와 human plan review gate
- implementation 전 policy-context sync gate
- Codex / Claude Code 실행 surface
- domain/spec/test/engineering/conversation trace review lanes
- feedback reopen flow
- decision memory 및 past-topic recall helper
- compact operator surface:
  - `doctor`
  - `topic-status`
  - `topics-status`

## 검증 포인트

- 자동 test / build 검증
- request-to-commit smoke flow
- multi-company sample repo dogfooding
- Shift AX launcher를 통한 real Claude code-edit 성공

## 포함되지 않은 것

- core workflow 단계로서의 GitHub PR automation
- 대형 dashboard / fleet UI
- webhook 중심의 장기 CI orchestration
