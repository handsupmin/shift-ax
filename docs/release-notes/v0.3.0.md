# Shift AX v0.3.0

## Highlights

- moved the primary reusable knowledge model to `~/.shift-ax/`
- added a structured global onboarding flow for work types, repositories, procedures, and domain language
- made request planning stop by default when the global index is missing
- added backup-first global knowledge updates
- made Codex use OMX-style `$...` command surfaces through `.codex/skills/*/SKILL.md`
- kept Claude Code on `/...` commands
- added a persisted shell preference for:
  - language
  - default full-auto mode
- mapped full-auto to runtime-specific flags:
  - Codex: `--yolo`
  - Claude Code: `--dangerously-skip-permissions`
- simplified the README into:
  - what Shift AX is
  - how to install it
  - how to use it
  - a prompt you can hand to another LLM
