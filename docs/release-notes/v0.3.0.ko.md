# Shift AX v0.3.0

## 핵심 변경

- 재사용 지식의 기본 저장소를 `~/.shift-ax/` 로 전환
- 작업 유형 / 레포 / 절차 / 도메인 언어를 모으는 글로벌 온보딩 추가
- 글로벌 인덱스가 없으면 request planning을 기본적으로 중단
- backup-first 글로벌 지식 갱신 추가
- Codex는 OMX 스타일의 `$...` 명령 surface 로 전환
  - `.codex/skills/*/SKILL.md`
- Claude Code는 `/...` 명령 유지
- 셸 전역 설정 저장 추가:
  - 언어
  - 기본 full-auto 모드
- full-auto를 런타임별 실제 플래그에 연결
  - Codex: `--yolo`
  - Claude Code: `--dangerously-skip-permissions`
- README를 더 단순하게 재작성
  - 이 도구가 뭐하는지
  - 어떻게 설치하는지
  - 어떻게 사용하는지
  - 다른 LLM에 넘길 프롬프트

- `npm test`
