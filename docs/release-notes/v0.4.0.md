# Shift AX v0.4.0

## Highlights

- upgraded implementation-plan artifacts with explicit acceptance criteria, verification commands, dependencies, likely files, checkpoints, execution tasks, and lightweight guardrails
- expanded execution handoff into a richer task contract with lane metadata, scoped paths, warnings, and clearer execution prompts while preserving the short=`subagent` / long=`tmux` split
- added compact welcome-back and readiness signals to `topic-status`, `pause-work`, and resume guidance so interrupted work can restart with less confusion and no dashboard overhead
- distinguished CI-sourced verification debt from local verification failures and tightened reproduce-first / evidence-not-instructions workflow rules
- added review playbooks for all five review lanes and refreshed Codex / Claude scaffolds to keep resume flows grounded in file-backed artifacts
