# Shift AX v0.3.2

## 핵심 변경

- 온보딩을 한 번에 다 묻는 방식에서 벗어나, 한 번에 하나씩 확인하는 단계형 심층 인터뷰 흐름으로 재설계
- 사용자가 언급한 레포를 먼저 살펴본 뒤 역할·레이어·집중 경로·숨은 규칙을 추론하고, 1/2/3 선택지로 빠르게 확인받도록 변경
- 생성되는 procedure 문서에 숨은 코딩 컨벤션과 아키텍처 의도 메모를 함께 저장
- Codex/Claude 온보딩 스캐폴드 안내를 강화해서 role/work/repo/glossary 불확실성이 사라질 때까지 티키타카가 이어지도록 정리
- overwrite 확인도 인터뷰의 다른 예/네 계열 응답과 같은 방식으로 받아서 한국어 온보딩 마찰을 줄임
