# Shift AX v0.1.0

## Summary

Shift AX v0.1.0 delivers the first complete request-to-commit control plane.

It is designed for teams that are new to AX and need:
- document-first grounding
- visible planning and review gates
- auditable topic artifacts
- safe defaults over workflow complexity

## Included in v0.1.0

- onboarding for domain, policy, and architecture context
- base-context discovery and glossary seeding
- request bootstrap into topic/worktree artifacts
- planning interview and human plan review gate
- policy-context sync gate before implementation
- execution orchestration and launch surfaces for Codex / Claude Code
- review lanes for domain/spec/test/engineering/conversation trace
- feedback reopen flow
- decision memory and past-topic recall helpers
- compact operator surfaces:
  - `doctor`
  - `topic-status`
  - `topics-status`

## Verification highlights

- automated test and build verification

- real Claude code-edit success through Shift AX launcher

## Not included

- GitHub PR automation as a core workflow stage
- large dashboard / fleet UI
- long-running webhook-driven CI orchestration

## Operator takeaway

v0.1.0 is ready for controlled team rollout when the pilot repo has:
- reviewed base-context docs
- green doctor output
- at least one real platform-backed execution success
