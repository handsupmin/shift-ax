# Base Context

`docs/base-context/` is the tracked home for organization knowledge that Shift AX may need during planning, implementation, review, and finalization.

This directory is intentionally broad. It can include:

- policy documents
- domain context
- architectural context
- engineering conventions
- company or team operating context

The top-level entrypoint is:

- [index.md](/Users/sangmin/sources/shift-ax/docs/base-context/index.md)

Shift AX should consult this index first, then load relevant detailed documents from this directory.
